# Teman Tuli 10-Week Execution Roadmap

## Weeks 1-2: Inclusive Discovery
- Interview Mahasiswa Tuli, hearing classmates, and accessibility-aware educators
- Validate classroom caption pain points and privacy concerns
- Lock live-caption and transcript archive MVP

## Weeks 3-6: Core Build
- Implement backend auth, private transcript sessions, timestamped segments, and feedback
- Implement iOS onboarding, live caption, transcript list, detail, notes, and settings
- Complete first full integration pass

## Weeks 7-8: Accessibility Polish
- Improve text size, contrast, empty states, and permission-denied states
- Test captioning in realistic noisy classroom scenarios
- Refine save flow so upload is explicit and understandable

## Week 9: Co-Creation Iteration
- Run prototype walkthrough with target users
- Capture accessibility feedback and before/after improvements
- Finalize privacy rationale and decision journal

## Week 10: Submission Packaging
- Finalize two case studies
- Finalize device test evidence and iteration notes
- Update CV and project links for submission
