# 3-Minute Demo Script (Teman Tuli)

## 0:00–0:20 — Problem
"Mahasiswa Tuli can miss important classroom context when discussion moves quickly or accessible notes are unavailable. Teman Tuli helps reduce that barrier."

## 0:20–1:10 — Product Walkthrough
- Register and open Live Caption
- Start live caption in Bahasa Indonesia
- Show large readable caption text
- Stop recording and save transcript privately

## 1:10–1:50 — Private Archive
- Open transcript list
- Read transcript detail
- Add personal notes
- Submit caption quality feedback

## 1:50–2:30 — Technical Depth
- Explain SwiftUI + Apple Speech + AVFoundation
- Explain backend: Fastify, Prisma, PostgreSQL, JWT
- Emphasize no automatic upload and private-by-default sessions

## 2:30–3:00 — Closing
"I built Teman Tuli as a simple but high-impact accessibility product aligned with SDG 4 and SDG 10, showing how technology can support more inclusive learning."
