# User Interview Round 2 — Prototype Validation

## Goal
Validate live-caption flow, private archive flow, and accessibility readability.

## Test Tasks
- Register and open Live Caption
- Start and stop captioning
- Save transcript privately
- Open transcript archive
- Add notes and submit caption-quality feedback

## Feedback Capture
- Was text readable from comfortable distance?
- Were privacy expectations clear?
- Did the save flow feel safe and understandable?
- Which caption mistakes are most harmful?
- What would make this usable in a real classroom?
