# Problem Statement & Hypothesis

## Context
Target users: Mahasiswa Tuli in Indonesian classroom settings.

## SDG Mapping
- SDG 4: Quality Education — improve access to classroom information.
- SDG 10: Reduced Inequalities — reduce communication barriers for Deaf/hard-of-hearing students.

## Problem Statement
Mahasiswa Tuli can miss important lecture context when conversations move quickly, interpreters are unavailable, or accessible notes are not provided consistently.

## v1 Hypothesis
If students can generate large live captions and save private transcript sessions only when they choose, then they will feel more included and more able to review class material after sessions.

## Success Signals
- Target users can start captioning without assistance.
- Users understand that transcripts are private and not uploaded automatically.
- Users report that saved transcripts help them recover missed class context.
