# Case Study A — Inclusive Discovery & SDG Impact

## Objective
Design a simple, high-impact accessibility tool that helps Mahasiswa Tuli access classroom conversations more independently.

## SDG Connection
- SDG 4: Improve access to learning material.
- SDG 10: Reduce barriers caused by communication access gaps.

## Process
1. Reframed project direction from mainstream productivity app to inclusive education.
2. Defined Mahasiswa Tuli as the primary user, not an edge case.
3. Chose privacy-first transcript saving to protect classroom trust.

## Outcome
Teman Tuli focuses on one powerful loop: live caption → explicit private save → review transcript later.

## What I Learned
- Accessibility products need trust as much as functionality.
- Simple MVPs can be high impact when the target user is specific.
- SDG impact is stronger when tied to a concrete daily barrier.
