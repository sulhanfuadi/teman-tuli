- SDG and problem framing
- inclusive design research plan
- co-creation interview guide
- iteration evidence
- privacy rationale
- case studies
- test matrix and execution runbook

2. Read `case-studies/case-study-b-technical-architecture.md`

## If Physical iPhone Is Unavailable

2. Use simulator evidence as MVP validation baseline for current scope
3. Schedule optional real-device expansion pass for higher runtime confidence
