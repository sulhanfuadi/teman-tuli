# Before / After UX Changes

## 1) Product Focus
- Before: General student productivity and wellbeing.
- After: Classroom captioning and private transcript archive for Mahasiswa Tuli.
- Evidence: Stronger SDG 4/10 relevance and clearer target user.

## 2) Data Flow
- Before: Routine productivity data uploaded through multiple flows.
- After: Transcript uploads only after explicit save action.
- Evidence: Privacy-first design improves trust.

## 3) Main Screen
- Before: Multi-tab task/focus/check-in dashboard.
- After: Large live-caption-first experience.
- Evidence: Accessibility MVP prioritizes readability and immediate class support.
